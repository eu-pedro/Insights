## 🚀 Começando



### 📋 Pré-requisitos


```
Ter o Node.js instalado em sua máquina.
```

### 🔧 Instalação

```
Obter o link do repositório do git
```

```
Em seu terminal de comando, executar  *https://github.com/eu-pedro/Insights.git*
```

```
Rodar um *npm install* e depois *npm run dev*
```
